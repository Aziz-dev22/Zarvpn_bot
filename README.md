# 🤖 ربات تلگرام فروش VPN (ZarVpn Bot)

سورس آماده و عمومی برای راه‌اندازی ربات هوشمند فروش اکانت‌های VPN متصل به پنل‌های Marzban و X-UI.

## 🚀 آموزش نصب سریع و خودکار روی سرور مجازی (Ubuntu)

کافیست با گوشی یا کامپیوتر وارد ترمینال سرور مجازی خود شوید و دستورات زیر را به ترتیب کپی و اجرا کنید. اسکریپت به صورت خودکار تمام مراحل را برای شما انجام خواهد داد:

```bash
# ۱. دانلود پروژه از مخزن گیت‌هاب شما
git clone [https://github.com/Aziz-dev22/Zarvpn_bot.git](https://github.com/Aziz-dev22/Zarvpn_bot.git)

# ۲. ورود به پوشه پروژه
cd Zarvpn_bot

# ۳. مجاز کردن و اجرای اسکریپت نصب خودکار
chmod +x install.sh && ./install.sh
